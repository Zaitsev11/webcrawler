The first paragraph: A <b>standard drink</b> is a notional drink that contains a specified amount of pure alcohol (<a href="/wiki/Ethanol" title="Ethanol">ethanol</a>). The standard drink is used in many countries to quantify alcohol intake. It is usually expressed as a certain measure of <a href="/wiki/Beer" title="Beer">beer</a>, <a href="/wiki/Wine" title="Wine">wine</a>, or <a href="/wiki/Distilled_beverage" title="Distilled beverage">spirits</a>. One standard drink always contains the same amount of alcohol regardless of the container size or the type of alcoholic beverage, but does not necessarily correspond to the typical serving size in the country in which it is served.
